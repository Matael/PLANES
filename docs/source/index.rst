PLANES --- Porous LAum NumErical Simulator
==========================================

PLANES is tool developped at the LAUM_ to simulate and compute the behaviour of acoustical systems involving porous
materials.

PLANES also provides a context to test hybrid methods for acoustical simulation.

    **Keywords :** porous material, numerical simulation, FEM, DGM, hybrid methods

.. _LAUM: http://laum.univ-lemans.fr
